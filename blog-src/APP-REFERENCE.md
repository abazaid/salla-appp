## ٣. جميع خدمات المنصة (٩ خدمات)

### ١. تحسين وصف المنتجات
- صياغة وصف احترافي منظم بعناوين وقوائم
- تحسين متوافق مع نية البحث والكلمات المفتاحية
- إمكانية التعديل اليدوي قبل الحفظ المباشر داخل سلة

### ٢. تحسين SEO المنتجات
- Meta Title و Meta Description محسّنة لكل منتج
- حفظ مباشر في متجرك على سلة

### ٣. تحسين SEO الماركات التجارية
- تحسين وصف و Meta Tags للماركات
- حفظ مباشر في سلة

### ٤. كاتب ALT للصور
- إنشاء ALT واضح ودقيق لكل صورة
- تحسين صورة واحدة، مجموعة صور، أو جميع صور المنتج
- تمييز الصور المحسّنة وغير المحسّنة مع سجل عمليات

### ٥. تحليل الكلمات المفتاحية
- تحليل الكلمة: حجم البحث، المنافسة، CPC، اقتراحات مرتبطة
- حفظ نتائج البحث السابقة

### ٦. تحليل سيو الدومين
- تحليل شامل للموقع مع مؤشرات ترتيب الكلمات
- مقارنة مع المنافسين

### ٧. سيو المتجر
- توليد صياغات قوية لصفحة المتجر الرئيسية
- تحسين Meta Title / Description / Keywords للمتجر
- تحديث فوري إلى إعدادات SEO في سلة

### ٨. كتابة مقالات المدونة (قريباً)
- مقالات متوافقة مع SEO جاهزة للنشر في مدونة سلة
- تركيز على كلمة مفتاحية رئيسية + كلمات طويلة داعمة
- ربط داخلي مع المنتجات والأقسام

### ٩. التحرير اليدوي (بدون AI)
- تعديل الوصف و Meta Tags يدوياً
- لا يستهلك من رصيدك
- حفظ مباشر في المتجر
